# Учёт доходов и расходов (Go + SQLite)

Локальный сервис для учета личных доходов и расходов с JSON API, хранением данных в SQLite и базовой аналитикой по категориям и бюджетам.

> Примечание: проект был разработан ранее как учебный pet-проект на Go и позже перенесен в этот аккаунт в рамках обновления и систематизации моего портфолио.

## О проекте

Этот проект — backend-приложение на Go для учета финансовых операций.  
Он позволяет создавать категории, добавлять транзакции, получать сводку за период, задавать лимиты по категориям и отслеживать превышение бюджета.

Проект был сделан как учебная практика по backend-разработке и работе с данными: HTTP API, маршрутизация, бизнес-логика, SQLite, тестирование и структура приложения на Go.

## Что умеет сервис

- создавать и получать список категорий;
- добавлять доходы и расходы;
- получать список операций за выбранный период;
- строить сводку по категориям;
- задавать лимиты бюджета;
- показывать предупреждения при превышении лимита;
- сохранять данные в SQLite.

## Основной функционал API

### Категории
- `GET /categories` — получить список категорий;
- `POST /categories` — создать новую категорию.

### Транзакции
- `POST /transactions` — добавить операцию;
- `GET /transactions?from=YYYY-MM-DD&to=YYYY-MM-DD` — получить операции за период.

### Сводка
- `GET /summary?from=YYYY-MM-DD&to=YYYY-MM-DD` — агрегаты по категориям:
  - доход;
  - расход;
  - итог;
  - количество операций.

### Бюджеты
- `GET /budgets` — получить список лимитов;
- `POST /budgets` — создать или обновить лимит по категории.

### Alerts
- `GET /alerts?from=YYYY-MM-DD&to=YYYY-MM-DD` — получить предупреждения о превышении бюджета.
